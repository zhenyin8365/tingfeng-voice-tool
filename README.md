# 庭锋装饰集团·AI语音工坊

基于阿里云百炼 CosyVoice v3.5-plus 的多员工 AI 语音合成工具。5位员工各自配置专属声音，输入文字即可合成语音，支持试听和下载。

## 与玉玉声音工坊的区别

| | 玉玉声音工坊 | 庭锋装饰集团·AI语音工坊 |
|---|---|---|
| 使用者 | 单人（玉玉） | 团队（5位员工） |
| 声音数量 | 1个音色 | 5个音色槽位 |
| API-Key | 单用户 | 全公司共享一个 |
| 快捷模板 | 通用场景 | 装修/装饰行业场景 |

## 技术栈

- **前端**：Streamlit（Python）
- **语音合成**：阿里云百炼 CosyVoice v3.5-plus REST API
- **声音复刻**：阿里云百炼控制台（一次性操作，免费）

## 页面功能

1. **配置**（管理员填一次，保存到 URL 参数，刷新不丢）
   - 阿里云百炼 API-Key（公司统一）
   - 声音1~5：每位员工的姓名 + 复刻音色 ID
2. **选择员工**：下拉切换当前使用谁的声音
3. **语音合成**
   - 自由输入文字（≤2000字）
   - 7 个装修行业快捷模板
   - 语速调节、音量调节
4. **合成结果**
   - 在线播放
   - 一键下载 MP3（文件名带员工名）

## 给新客户部署

1. 在百炼控制台帮每位员工复刻声音 → 拿到 5 个音色 ID
2. 打开部署页面，填好 API-Key 和 5 组员工+音色
3. 点「保存全部配置」，所有数据写入 URL 参数
4. 把带参数的链接发给客户公司
5. 员工打开即用，下拉选自己名字，无需登录、无需安装

## API 调用方式

```
POST https://dashscope.aliyuncs.com/api/v1/services/audio/tts/SpeechSynthesizer
Authorization: Bearer {api_key}
Body: {"model":"cosyvoice-v3.5-plus","input":{"text":"...","voice":"xxx","format":"mp3"},"parameters":{"speech_rate":1.0,"volume":80}}

→ 返回音频 URL → GET 下载 → MP3
```
